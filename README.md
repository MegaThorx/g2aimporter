# G2AImporter

[![Software License][ico-license]](LICENSE.md)

Simple G2A importer to import products to your own database.

## Install

Via Composer

``` bash
$ composer require zborowski/G2AImporter
```

## Usage

``` php
php artisan vendor:publishs
php artisan migrate
php artisan g2a:import
```
